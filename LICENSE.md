2. **User Copyright** (as the configuration owner)

- `Core/Src/main.cpp` (template, modified by user)

- `Core/Startup/startup_stm32h723zgtx.s` (ARM assembly startup)
- `Drivers/STM32H7xx_HAL_Driver/Src/` (auto-populated HAL sources)
- `USB_DEVICE/App/usbd_*` (auto-generated USB hooks)
- `MXC-03.ioc` (CubeMX project file)

### License Treatment

CubeMX-generated code is treated as follows:

- **Unmodified sections** (auto-generated only) → Subject to STMicroelectronics SLA0068
- **USER CODE sections** (user edits within `/* USER CODE BEGIN */...`/* USER CODE END */`) → Own copyright (BSD 3-Clause)
- **Modified non-USER CODE sections** → Hybrid: STMicroelectronics copyright + BSD-compatible contributor rights

### Regeneration Policy

When regenerating project from `.ioc` file:

1. All `USER CODE` sections are preserved
2. Auto-generated sections are overwritten with new CubeMX output
3. User modifications outside `USER CODE` blocks will be lost on regeneration

**Recommendation:** Keep all custom logic within marked `USER CODE` sections to survive CubeMX regeneration.

## 自動生成コード（CubeMX）

### 適用範囲

以下のファイルは **STM32CubeIDE/CubeMX により自動生成** され、以下の両方に従います：

1. **STMicroelectronics 著作権**（ジェネレータツール所有者として）
2. **ユーザー著作権**（設定所有者として）

対象ファイル：

- `Core/Inc/main.h`, `stm32h7xx_*.h`（ハードウェア設定ヘッダ）
- `Core/Src/main.cpp`（テンプレート、ユーザー修正）
- `Core/Src/stm32h7xx_hal_msp.c`, `stm32h7xx_it.c`, `system_stm32h7xx.c`
- `Core/Startup/startup_stm32h723zgtx.s`（ARM アセンブリスタートアップ）
- `Drivers/STM32H7xx_HAL_Driver/Src/`（自動入力 HAL ソース）
- `USB_DEVICE/App/usbd_*`（自動生成 USB フック）
- `MXC-03.ioc`（CubeMX プロジェクトファイル）

### ライセンス処理

CubeMX 生成コードは以下のように処理されます：

- **未修正セクション**（自動生成のみ）→ STMicroelectronics SLA0068 対象
- **USER CODE セクション**（`/* USER CODE BEGIN */`...`/* USER CODE END */` 内のユーザー編集）→ 独自著作権（BSD 3-Clause）
- **修正された非 USER CODE セクション** → ハイブリッド：STMicroelectronics 著作権 + BSD 互換貢献者権

### 再生成ポリシー

`.ioc` ファイルからプロジェクトを再生成する場合：

1. すべての `USER CODE` セクションが保存される
2. 自動生成セクションは新しい CubeMX 出力で上書きされる
3. `USER CODE` ブロック外のユーザー修正は再生成時に失われる

**推奨:** すべてのカスタムロジックを標識付き `USER CODE` セクション内に保持して CubeMX 再生成後も保存されるようにしてください。
